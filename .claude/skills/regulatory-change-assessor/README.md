# Regulatory Change Assessor Skill

## Overview

The **regulatory-change-assessor** skill systematically evaluates the impact of new or amended regulations on the Risk Taxonomy framework. It provides a structured approach to identifying gaps, affected artefacts, and implementation requirements.

## Quick Start

### Trigger Phrases

- "Assess regulatory impact of [regulation]"
- "What needs to change for [regulation]?"
- "Gap analysis for [regulation]"
- "Implementation requirements for [regulation]"
- "What artefacts need updating for [regulation]?"
